CSS Tutorial

CSS គឺជាភាសាដែលយើងប្រើដើម្បីរចនាឯកសារ HTML ។

CSS ពិពណ៌នាអំពីរបៀបបង្ហាញធាតុ HTML ។

ការបង្រៀននេះនឹងបង្រៀនអ្នកពី CSS ពីកំរិតមូលដ្ឋានដល់កំរិតខ្ពស់។

Examples in Each Chapter

ឯកសារបង្រៀន CSS នេះមានឧទាហរណ៍ CSS រាប់រយ។

ជាមួយកម្មវិធីនិពន្ធតាមអ៊ិនធឺរណែតរបស់យើងអ្នកអាចកែសម្រួល CSS ហើយចុចលើប៊ូតុងដើម្បីមើលលទ្ធផល។

Example

body {
background-color: lightblue;
}
h1 {
color: white;
text-align: center;
}
p {
font-family: verdana;
font-size: 20px;
}
